Tipos de texto en HTML

En HTML podemos encotrar distintos tipos de texto los cuales veremos continuacion.

Encabezados(heads).

HTML cuenta con un listado de encabezados(heads) los cuales son de mucha utiliza para dar un cierto nivel de importancia a nuestros titulos en una pagina web.

Entre los encabezados(heads) podemos encotrar la siguiente lista segun su nivel de importancia.

  • h1
  • h2
  • h3
  • h4
  • h5
  • h6

Parrafos

En HTML podemos definir parrafos los cuales se de definen por medio de la etiqueta < p > < /p >

Ejemplo de un parrafo usando la etiqueta p

< p >Lorem ipsum dolor sit amet, consectetur adipisicing elit.
Magni, sed officia veniam minima nobis ipsum quisquam mollitia alias placeat
nulla numquam veritatis, quasi, accusantium perspiciatis nemo. Minima minus maiores hic! < /p >

Saltos de linea

Un salto de linea puede ser definido por la etiqueta < br > esta etiqueta es solo de apertura no se incluye la etiqueta de cierre

Espacios en blanco

Para poder agregar un espacio en blanco adicional al que HTML nos da se escribe lo siguiente &nbsp

Negritas

Para poder escribir textos en negritas lo podemos hacer por medio de la etiqueta:
< b >Este texto esta en negritas< /b >.

Cursivas

Para poder escribir textos en cursiva utilizamos la etiqueta:
< i > Este es un texto en cursivas < /i >

Resaltado de Textos

Para resaltar textos utilizamos la etiqueta:
< strong > Este es un texto resaltado< /strong >

Enfasis en textos

Para poder utilizar enfasis en texto utilizamos la etiqueta:
<em > Este es un texto en enfasis < /em >

Seleccion de textos

Para poder seleccionar un texto utilizamos la etiqueta span a la cual le definimos un color para resaltar la seleccion:
<span > Este es un texto seleccionado < /span >

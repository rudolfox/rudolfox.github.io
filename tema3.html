Enlaces en HTML

En HTML existen tres tipos de enlaces los cuales veremos a continuacion.

  • Relativo

    • El enlace relativo me toma el url y me lleva a un nuevo contenido web.

  • Absoluto

    • El enlace relativo conserva el url y me carga sobre un nuevo contenigo web sobre el mismo url.

  • Contenido de Descarga

    • Sirve para descargar algun documento archivo o imagen de interes

El enlace en HTML se define mediante la etiqueta <a> </a> la cual trabaja con el
atributo href el cual nos ayudara a definir el tipo de enlace con el que se trabajara.

Un enlace en html toma la siguiente forma.

Ejemplos

<a href="estrutura.html" > Ejemplos de estructura html < a >

<a href="Temario.pdf" download="" >Temario del curso (descarga) < a >

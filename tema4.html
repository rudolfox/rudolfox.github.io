Listas

Las listas en HTML son colecciones de elementos de mismo tipo.

HTML cuenta con los siguientes tipos de lista:

  • Lista ordenada
  • Lista desordenada
  • Lista de descripcion

Ejemplo con listas

Lista ordenada (ordered list)

Para una lista ordena utilizamos la etiqueta < ol >____< /ol > y toma la siguiente forma:

< ol >
    < li >E1 < /li >
    < li >E2 < /li >
    < li >E3 < /li >
< ol >

En donde E1...E3 son los elementos que contendra la lista y < li >E1 < /li >
la etiqueta que define el item o elemento dentro de la lista.

Lista desordenada (unordered list)

Para una lista desordena utilizamos la etiqueta < ul >____< /ul > y toma la siguiente forma:

< ul >
    < li >E1 < /li >
    < li >E2 < /li >
    < li >E3 < /li >
< ul >

En donde E1...E3 son los elementos que contendra la lista y < li >E1 < /li >
la etiqueta que define el item o elemento dentro de la lista.

Lista de descripcion

Una lista de descripcion consiste en un conjunto de descripciones listadas.

< d1 >
    < dt > HTML < /dt >
    < dd >Lenguaje para explicar el contenido de una pagina web < /dd >

    < dt > CSS < /dt >
    < dd >Lenguaje para especificar la apariencia de una pagina web < /dd >

    < dt >JS < /dt >
    < dd > Lenguaje para el comportamiento de una pagina web < /dd >
<d1 >

En donde podemos observar el uso de las siguientes etiquetas:



< d1 >_____< /d1 > Etiqueta que define el cuerpo de la lista de descripcion

< dt >_____< /dt > Etiqueta que define el titulo en nuestra lista de descripcion

< dd >_____< /dd > Etiqueta que define el contenido de la descripcion de la lista
